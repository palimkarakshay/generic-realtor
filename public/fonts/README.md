# Self-hosted fonts

These are intentionally empty placeholders. Before deploying, replace them with the actual variable WOFF2 files.

## Where to get them (both Open Font Licensed — free for commercial use)

**Fraunces** (display serif)
- https://github.com/undercase/Fraunces
- Download the variable WOFF2 from the latest release
- Save as `Fraunces-Variable.woff2` in this directory

**Public Sans** (body sans-serif)
- https://github.com/uswds/public-sans
- Download the variable WOFF2 from the latest release
- Save as `PublicSans-Variable.woff2` in this directory

## Why self-host?

- No Google Fonts tracking (better privacy)
- No third-party request (faster initial paint)
- Caches cleanly with Cloudflare's free tier
- Variable fonts ship one file for every weight from 100 to 900
